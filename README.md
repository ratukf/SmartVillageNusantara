# SmartVillageNusantara

SmartVillageNusantara is a **static multi-page website** project built by translating a Figma design into a fully responsive web interface.  
The main focus of this project is **UI accuracy**, **layout consistency**, and **responsive behavior** across different screen sizes.

## Project Purpose

This project was created to:

- Convert a Figma UI design into a real, functional website.
- Practice clean HTML structure and organized styling.
- Ensure responsiveness on desktop, tablet, and mobile devices.
- Serve as a frontend prototype, portfolio piece, or static website template.
